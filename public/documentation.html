This is a work in progress for my Movie API


Request URL HTTP Method Request body format data Request body format data
Get all list of Movies /movies GET None A JSON object holding data about all the movies
Get data about (description, genre, director, URL, whether it's featured or not) about a single movie by title to the user GET /movies/[Title] /movies/:Title None A JSON object holding data about a single movie containing a title, description, genre, director, image, and featured

Example:
{ "title": "Avengers: End Game", "description":"Avengers: End Game came out 2019, this is based off of the characters in the Marvel Universe", "genre": "Action/scifi", "director": "Joe Russo & Anthony Russo", "image": "", "featured": "true" }
Return data about genre (description) by name/title (e.g. "Thriller") GET /movies/genre/[genre] GET /movies/genre/:name None
Return data about a director (bio, birth year, death year) by name GET /movies/directors/[name] /movies/directors/:name None JSON holding data about director
Allow new users to register POST /users None A JSON using data for registration
Allow users to update their user info(username) PUT /user/[username] /user/:username JSON holding data about the users information
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) POST /users/[username]/movies/[movieID] /users/[username]/movies/:movieID None A JSON object holding data about all your favourite movie
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) DELETE /user/[username]/movies/[movieID] /user/[username]/movies/:movieID A JSON object hooding data about the current favorite list
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) DELETE user/[name] user/:name None Sends message of account deleted
